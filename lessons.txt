ls - перегляд файлів всередині директорії
    -l - файли з нового рядка в довгому форматі
more [file] - перегляд змісту файлу
mkdir [dir] - створити директорію
mv [file] [dir] - перемістити файл/директорію в директорію
mv [file] [file] - перейменувати файл/директорію
cd [dir] - зманити директорію
pwd - друк робочого каталогу


/[dir]/.../[file] - абсолютний шлях (з корня)
[dir]/.../[file] - відносний шлях (з поточної директорії)
.. - попередній каталог в відносному шляху
cp - копіює файл
    -r - копіює каталог (зі структурою всередині)
rm - видаляє файл
    -r - видаляє директорію з усіма файлами
rmdir - видаляє директорію (порожню)

Довгий формат файлу:
    | -rw-r--r-- keeper prim 485 9:34 sq_monks  |
    | ^    ^       ^     ^    ^   ^        ^    |
    | type |     owner   |   size |       name  |
    |   security       group   last mod         |
Тип: файл(-) / каталог(d)
Безпека включає 9 символів (3 групи по 3 символи):
    r - читати
    w - записати
    x - виконати (для програм)
    - - заборона дії
    rwx     rwx     rwx     - всі дії дозволені для всіх
    user   group   other

chmod - зміна дозволів безпеки (+ дозвіл / - заборона)
    u-x [file] - заборонити виконання файлу для user (owner)
    ugo+rwx [file] - дозволити всім всі дії

При введенні імен файлів чи каталогів можна використовувати:
    * - будь-яка кількість будь-яких символів;
    ? - один будь-який символ

groups - виведе список груп, до яких належить користувач


~ - скорочення домашнього каталога в шляхові (свого)
~[name] - скорочення домашнього каталога користувача [name]

man - документація по системі Linux (manual)
man [com] - справка по команді [com]
man -k [keyword] - видасть весь перелік команд, з *[keyword]*
finger - програма пошуку інформації по користувачам
find [way] -name [name] - пошук файлу [name], розташованого в [way]
    "poem*" - знайде всі файли з назвою "poem..." (вимагає [" "])
cat [name1] [name2] ... - об'єднує перераховані файли на екран
    > [final] - перезапише файл [final] результатом операції
    >> [final]

lpr [file] - відправити [file] на принтер за вмовчуванням
    -P [prName] - відправити на принтер [prName] (+для lpq, lprm)
lpq - відображення чарги друку
lprm [job] - видалити з черги друку завдання [job]


df - виведе кількість дискового простору

ps - статус процесу
    aux - детальний список усіх процесів
[command1] | [command2] - надсилає результат [command1] в [command2]
grep [temp] - знайти кожне повторення шаблону в каналі (після "|")
    [file] - знайти кожне повторення шаблону в файлі
ps aux | grep [key] - виведе всі процеси, що містять [key]
kill [PID] - припинити процес з ідентифікатором [PID]
    -9 - примусове негайне завершення
